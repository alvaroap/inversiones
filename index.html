Editar activos
Los activos se definen en datos.json, el archivo que tienes en GitHub. Edítalo y vuelve a subirlo para que el dashboard lo cargue.
{
  "id": "mi_activo",
  "nombre": "Nombre visible",
  "ticker": "BME:XXXX",
  "color": "#c8f135",
  "capitalInicial": 0,
  "desdeMes": "Ene",
  "hastaMes": "Dic"
}
Meses válidos: Ene · Feb · Mar · Abr · May · Jun · Jul · Ago · Sep · Oct · Nov · Dic
Para activo parcial: "desdeMes": "Mar", "hastaMes": "Jun"
Tras editar: sube datos.json a GitHub y recarga la página
